LINKED-LIST

Introduction=

Linked-list is a linear data-structure.It is a collection of objects known as a node.It work with Dynamic memory allocation,it stored element anywhere in the memory or randomly stored elements.Linked-list elements are dependent on each other,elements are linked to each other through pointers.

Defination=

Linked-list is liner collection of data elements called NODES.Each node is divided into 2 or more parts,the first part is data parts that contain tha keyvalue and second part is pointer part that contain the pointer to next node.
Example:-

types of Linked-list:-

  1. singly Linked-list=

    It is commonly used linked-list.It's elements contain only two parts,one part is for data that contain keyvalue and second part is pointer part that contain address of next node.
    Node syantax:-

    Example:-

    Representation:-
    struct node
    {
            int data;
        struct node *next;
    };

  2. Doubly Linked-list=

    It is linear data-structure ,it's elements contain three parts,one part is for data that contain keyvalue and two other parts are pointer parts that conatain address of previous node and next node.
    Node syantax:-

    Example:-

    Representation:-
     struct node
    {    
        struct node *prev;
        int data;
        struct node *next;
    };

  3. Circular Linked-list=

    It is similar to single linked-list,in this type of linked-list the last node hold tha address of first node.we can traverse in any direction.
    Example:-

    Representation:-
    struct node
    {     
        int data;
        struct node *next;
    };

  4. double Circular Linked-list=

    It has both feature of circular linked-list and doubly linked-list.In this each node contain the keyvalue,previous node's address and last node's address and the last node holds the address of first node.
    Example:-

    Representation:-
    struct node
    {    
        struct node *prev;
        int data;
        struct node *next;

    };

Advantages and Disadvantages of Linked-list

    Advantages of Linked-list:-

  1. Linked-list is Dynamic data-structure.
  2. Insertation and Deletation is easy in Linked-list.
  3. Memory utilization is efficient in Linked-list.
  4. Stack and Queue can be implement using a Linked-list.
.

    Disadvantages of Linked-list:-

  1. The element in Linked-list occupy more memory.
  2. Traversal is not easy.
  3. Randomly access is not allow.
  4. Linked-list's elements are dependent on each other.
